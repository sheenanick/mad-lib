# Mad Libs

#### 08.10.2016

#### By **Sheena Nickerson** and **Katrina Hirsch**

## Description

This is a website where a user can play Mad Libs. The goal of this project was to practice using jQuery elements with HTML forms.

## Setup/Installation Requirements

* `git clone https://github.com/sheenanick/mad-lib`
* Open the html file with the browser of your choice
